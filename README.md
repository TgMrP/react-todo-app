# React Todo List

Simple application to perform. You can add a to-do, this list is saved in a cookie. And the desired elements can be deleted from the list or marked as complete.

#

## install with yarn

- [ ] clone the repo
- [ ] `yarn install`
- [ ] `yarn start`
- [ ] go to http://localhost:3000

#

## install with npm

- [ ] clone the repo
- [ ] `npm install`
- [ ] `npm start`
- [ ] go to http://localhost:3000
